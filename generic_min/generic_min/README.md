# 1. Introdução

Este projeto possui 4 pastas com formato `version_X`, sendo `X` um número de `0` à `3` representando versões sucessivas da função `min()` desenvolvidas durante as 2 aulas sobre programação abstrata.

Em cada pasta você vai encontrar um arquivo `README.md` com explicações e instruções sobre a versão em questão.

O projeto na pasta `version_qsort` simplesmente espera que você use a função [`qsort()`](https://en.cppreference.com/w/cpp/algorithm/qsort) para ordernar o vetor de inteiros e o vetor de alimentos.
